Complementary descriptions to some scientific papers (prior to 2005) and ExtSym software page

Purpose: The list to the right contains complementary descriptions of some scientific works I contributed to prior to 2005. These are (except ExtSym) meant to be short, easier-than-article-to-read entry points to such works. The page for ExtSym came later, and was created as a landing and download page used with the publication of this software (note this was before platforms such as GitHub became available). For other work I have contributed to, see ResearchGate and Google Scholar.
